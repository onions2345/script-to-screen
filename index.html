Script to Screen

Paste a script. Gemini works out who speaks each line and casts a voice, reads it aloud, and sketches the visual beats — then stitches the voices into one audio track. Runs entirely in your browser.

Setup & keys
No key yet? Get a free one from Google AI Studio — sign in with Google, click "Create API key," paste it here. Each visitor uses their own key; keys can't be handed out by a login button.
Your key stays in this browser only (saved to localStorage on this device). Never commit it to the repo — GitHub Pages is public. For safety, restrict the key to your Pages URL in Google Cloud Console → Credentials → HTTP referrers.
Model names change. If one 404s, try newer IDs: TTS gemini-3.1-flash-tts-preview, image gemini-3.1-flash-image-preview, text gemini-3-flash. The free tier is generous but rate-limited (~10 image calls/min).
